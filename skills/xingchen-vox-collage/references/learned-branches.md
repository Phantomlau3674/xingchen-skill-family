## What To Reject

- Provider or model IDs as core workflow logic.
- Treating two flattened endpoints as guaranteed layer control.
- Essential text baked into generated posters.
- Fixed scene types selected by array index.
- A generic torn-paper card around every object.
- Audio, research, or packaging complexity used to hide weak visuals.
- Repository popularity as proof of render quality.

## Reconstructed Causal Grammar

Use this sequence:

`scene job -> visual proposition -> style DNA -> settled hero frame -> independent actors -> deterministic motion -> rendered evidence -> revision`

For short metaphors, an optional generated-video route may replace deterministic motion only after the scene proves it has no exact text, proof, identity, count, or endpoint requirement.

Do not copy Vox logos, proprietary title packaging, exact layouts, private prompts, or source assets.
