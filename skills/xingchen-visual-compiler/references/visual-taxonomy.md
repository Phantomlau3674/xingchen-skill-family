# Visual Taxonomy (Fallback Appendix)

Do not use this file as the first decision layer in Xingchen V2.

First derive scene forces, proof visibility, motion verbs, and attention direction.

Use this appendix only when:

- the project explicitly asks for a stable fallback
- the direction layer flags `fallback_strategy`
- recovery work must land safely under time pressure

Fallback scene labels:

- `statement`
- `comparison`
- `infographic`
- `proof-card`
- `architecture`
- `timeline`
- `quote`
- `terminal`

These labels are recovery helpers, not the source of creative truth.
