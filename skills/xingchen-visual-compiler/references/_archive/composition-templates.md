# Composition Templates (Fallback Appendix)

Do not start new Xingchen V2 scene direction from this file.

Use these templates only when a scene enters explicit fallback mode and still needs a clear, stable layout answer.

## Fallback Rules

- one dominant anchor only
- leave subtitle-safe room without shrinking proof-critical pixels
- use the design system for spacing and scale
- one side must win in a comparison scene

## Available Fallback Layouts

- `statement`: one idea, one anchor, minimal support text
- `comparison`: one winner, one loser, one conclusion
- `infographic`: structure first, screenshot second
- `proof-card`: proof pixels stay readable
- `architecture`: explain relations, not vibes
- `timeline`: process order is the point
- `quote`: wording is the hook
- `terminal`: operational rhythm, not decorative typing
