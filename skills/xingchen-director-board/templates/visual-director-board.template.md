## Tech Stack Policy

| Stack | Allowed Use | Forbidden Use | Preview Requirement |
|---|---|---|---|
| Remotion | Timeline, subtitles, proof overlays, component scenes, final composition | Generic visual replacement for source understanding | Local Remotion preview or still+timing evidence |
| HTML 3D | Camera-as-argument, concept room, proof plane, 3D card stack | Generic premium background | Browser preview or Remotion Three preview with camera/depth proof |
| Hyperframes | HTML/canvas candidate scene | Full-video controller | Candidate artifact then Remotion/HTML promotion path |
| Spark | 3DGS, procedural splat, world plate, spatial traversal | Ordinary high-end background | SparkRoutePreview or equivalent browser canvas plate |
| VibeMotion | Candidate motion source or transparent layer | Final controller | Real candidate artifact and promotion target |
| Source media | Original recording/screenshot/clip as evidence | Decorative texture that hides proof | Legibility/crop/redaction check |
| Gen insert | Hero realism gap | Default full-video generator | Source reason and approval boundary |
