# Local Stitch reference manifest

The public Xingchen distribution keeps only a provenance manifest for four local Stitch reference packs. The ZIP archives are deliberately not redistributed because their source package did not establish public redistribution rights.

The family names and hashes may be used to recognize a separately obtained local pack. Do not treat this manifest as a download index, source-code license, template library, or permission to copy archived HTML, CSS, remote image URLs, brand text, or layouts.

If a user independently has a licensed pack, keep it outside the public skill tree and configure a local reference path. Xingchen should still rewrite the visual question through the current director board and deterministic implementation contracts.
