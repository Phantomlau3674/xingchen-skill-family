# Meta Concept Examples

These are examples of form-to-content binding. They are not a menu.

## Positive Examples

### skill-run

- thesis: agent 与 skill 让内容生产从手工编辑变成可执行流程
- meta_concept: 这条视频本身，就是一次 skill 运行
- chrome_family: terminal-run
- motion_signature: type, blink, diff, advance
- anti_reference: reject `saas-keynote` because the proof is execution, not decoration

### data-journal

- thesis: 论点本身来自数据变化，不来自态度宣言
- meta_concept: 图表不是配图，而是观点正在变形
- chrome_family: data-journal
- motion_signature: count, annotate, morph, lock
- anti_reference: reject `generic-gradient-particles` because spectacle without numeric causality breaks trust

### documentary-contact

- thesis: 观众必须相信证据真的存在，而且就在眼前
- meta_concept: 这不是讲解证据，而是在现场翻证据
- chrome_family: documentary-contact
- motion_signature: scan, pause, mark, timestamp
- anti_reference: reject `thumbnail-evidence` because tiny proof kills the claim

### kinetic-type

- thesis: 文本本身就是冲突和节拍
- meta_concept: 口播的力量直接长成排版，不借外置 UI 代替
- chrome_family: kinetic-type
- motion_signature: slam, stagger, hold, flash
- anti_reference: reject `narration-echo-bubble` because the screen must hit, not repeat

### studio-process

- thesis: 做视频的过程就是内容的一部分
- meta_concept: 观众看到的是制作现场在自己暴露工作痕迹
- chrome_family: studio-process
- motion_signature: scrub, stack, preview, reorder
- anti_reference: reject `pseudo-dashboard-card` because fake product panels dilute making-as-content

### field-log

- thesis: 这是一次真实发现过程，而不是发布会
- meta_concept: 整条片子像一次现场记录，线索被逐步挖出来
- chrome_family: field-log
- motion_signature: uncover, pin, note, tag
- anti_reference: reject `apple-keynote-sans` because polished symmetry kills discovery feeling

## Negative Examples

### saas-keynote

- thesis: anything
- meta_concept: 现代科技感黑底大字卡片
- chrome_family: keynote-modern
- motion_signature: glow, float, fade
- anti_reference: this is the anti_reference itself; it severs form from content and defaults to slideware

### generic-gradient-particles

- thesis: anything
- meta_concept: 高级感靠粒子和扫光撑起来
- chrome_family: gradient-particles
- motion_signature: drift, sparkle, haze
- anti_reference: reject because it adds atmosphere but no argument

### meme-overload

- thesis: anything
- meta_concept: 让节奏靠梗图密度赢
- chrome_family: meme-wall
- motion_signature: pop, shake, crash-zoom, sticker
- anti_reference: reject because the viewer remembers the jokes, not the proof

### narration-echo-bubble

- thesis: anything
- meta_concept: 屏幕上的大气泡替旁白再读一遍
- chrome_family: echo-bubble
- motion_signature: float, pulse, linger
- anti_reference: reject because screen text should support the scene, not duplicate the ear
