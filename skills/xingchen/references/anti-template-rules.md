# Anti-Template Rules

These are hard bans for Xingchen V2.

## Never Default To

- shrinking proof into a corner card when the claim needs readable pixels
- fake dashboard chrome that does not belong to the project
- four competing elements on one scene with no dominant anchor
- long on-screen text that simply repeats narration
- perfectly symmetric comparisons with no winner
- scene reuse that makes different videos feel like the same template with new words

## Smell Tests

If any of these are true, the scene is probably templated:

- you can swap the text and nothing else would need to change
- the motion looks equally plausible for any topic
- the headline, screenshot, and subtitle all compete for first read
- the scene has style, but no reason for that style
